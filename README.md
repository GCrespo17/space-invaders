# space-invaders

## Description
Project to learn c++

## Compile instructions:
 
 1. Install SFML

 2. To compile the project
 
 In the root folder of your project

 ```console
 mkdir build
 cd build
 cmake ..
 cmake --build .
 ```

## To make your own build

In this case, i used Ninja

```console
mkdir build
cd build
cmake -G Ninja -S ..
```
